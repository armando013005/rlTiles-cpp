# rlTiles-cpp
A tile map reader and renderer for raylib

# Building
rlTiles-cpp is setup to use premake to generate a static library and example.

## For Visual Studio
Run premake-VisualStudio.bat

## For MinGW-W64
Run premake-mingw.bat

## For GCC On Linux
Run premake5 gmake2

## For MacOSX
Run premake5.osx gmake2
